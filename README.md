# Java Study Repository

This repository contains Java study materials and practice examples from coursework at **Nha Trang University**, under the academic guidance of **PhD. Mai Cường Thọ**.

## Contents

- Core Java concepts
- Object-Oriented Programming (OOP)
- Practice exercises and examples
- Basic software design principles

## Purpose

The repository is intended to support learning and practice for students studying Java at the university level.

## Technologies

- Java (JDK)
- IntelliJ IDEA (IDE)
